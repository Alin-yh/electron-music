#### 安装

``` bash
# 安装项目所需要的依赖(第一次有点慢，耐心点)
npm install

# 运行项目 serve with hot reload at localhost:9080
npm run dev

# 打包项目(会在build目录下有exe安装程序和绿色版的程序) build electron application for production
npm run build

```
